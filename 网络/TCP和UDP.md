**TCP（传输控制协议）和 UDP（用户数据报协议）的主要区别如下**

| 比较项目 | TCP                      | UDP                      |
| ---- | ------------------------ | ------------------------ |
| 连接性  | 面向连接                     | 无连接                      |
| 可靠性  | 提供可靠的数据传输，确保数据无差错、按顺序到达  | 不保证数据的可靠性，可能出现数据丢失、乱序等情况 |
| 数据顺序 | 保证数据按发送顺序到达              | 不保证数据顺序                  |
| 流控   | 有流量控制机制                  | 无流量控制机制                  |
| 拥塞控制 | 有拥塞控制机制                  | 无拥塞控制机制                  |
| 头部开销 | 较大                       | 较小                       |
| 应用场景 | 对数据准确性要求高的应用，如文件传输、电子邮件等 | 对实时性要求高的应用，如视频会议、直播等     |
